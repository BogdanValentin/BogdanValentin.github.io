# BogdanValentin.github.io

## New website
A personal photography portfolio built with vanilla HTML, CSS & JavaScript.

**Features:**
- Responsive photo gallery with dynamic grid layout (adapts from 2×3 to 8×12 based on photo count)
- Category-based browsing (Animals, Architecture, Landscapes, etc.)
- Lightbox viewer with image details
- Mobile-friendly with hamburger menu navigation
- About & Contact panels
- Preloader animation
- Photo optimization script (`optimize-photos.sh`)

**Tech stack:** HTML · CSS · JavaScript (no frameworks)

**Live:** [bogdanvalentin.github.io](https://bogdanvalentin.github.io)
